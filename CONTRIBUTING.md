# Contributing

This is a small, casual community project made by Agent YellowLightning. Contributions are welcome, but please keep things lightweight and pragmatic.

## What We Accept
- Rebuilds of Scanner graphics (SVG/PNG/EPS/PSD/...).
- Editable templates (SVG/AI/EPS/PSD/...).

## How Contributions Work
- Please use **Issues** for requests, missing assets, or questions.
- Use **Pull Requests** for actual contributions.

## File Standards
- Filenames: `snake_case`.
- Variants in name: `icon_portal_attack_resistance.png`.
- Vector preferred: `SVG` required.
- Optional extras: `AI`, `EPS` or `PNG`.

## Legal
See `README.md` and `LICENSE` for the legal and licensing notes. Please only contribute work you created.
